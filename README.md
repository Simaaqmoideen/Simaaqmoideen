### 🌟 Featured Project

**[CareerSync — Smart Campus Placement Ecosystem](https://github.com/Simaaqmoideen/careersync1)**
*A full role-based placement platform (Admin / Student / Alumni) built for a hackathon.*

- 🤖 **AI Resume Analyzer** — Gemini-powered ATS scoring and suggestions
- 📊 **Smart Eligibility Engine** — auto-filters students by CGPA, branch, backlogs
- 🎯 **Skill Gap Prediction** — compares your profile against placed students
- 💬 **Context-aware Career Chatbot** using live placement-drive data
- 🔐 Prepared statements, CSRF protection, bcrypt hashing, session regeneration
- 🔗 Live demo: [careersyncer-delta.vercel.app](https://careersyncer-delta.vercel.app)

> This is the project that best represents what I can build end-to-end — pin it at the top of your repos tab if it isn't already.
